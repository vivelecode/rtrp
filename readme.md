Real-Time Renderer (shaders)
===========================

Краткое описание
---------------------------

Разработанная система рендеринга предназначена для получения результирующего кадра 
растрового изображения в процессе его построения согласно 3D модели в реальном времени 
и состоит из двух модулей:

	* независимо компилируемый программный модуль (плагин) для Autodesk 3Ds MAX 2013;
	* графическое приложение на базе объектно-ориентированного графического движка с открытым исходным кодом OGRE.
	
При помощи плагина пользователь осуществляет экспорт собственной трехмерной сцены из 3D редактора и предварительную
настройку графического приложения. Плагин извлекает информацию о вершинах, материалах, текстурах источниках света в сцене,
а также настройках камеры и осуществляет их предварительную подготовку для дальнейшего использования графическим 
программным ядром путем. Затем происходит старт связанного стороннего приложения с параметрами, значения которых 
пользователь изменяет по мере необходимости в интерфейсе плагина.

Графическое приложение, получив информацию о настройках и местонахождении ресурсов экспортируемой сцены, осуществляет
загрузку данных и формирование 3D сцены, после чего, пользователь получает сгенерированный результирующий кадр 
изображения проекции сцены и управление основной камерой.
